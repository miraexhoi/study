# Socket  
- 웹 소켓 핸드쉐이크
> new WebSocket(url)을 호출해 소켓을 생성하면 즉시 연결이 시작됩니다.  
커넥션이 유지되는 동안, 브라우저는 (헤더를 사용해) 서버에 '웹소켓을 지원하나요?' 라고 물어봅니다.  
이에 서버가 '네’라는 응답을 하면 서버-브라우저간 통신은 HTTP가 아닌 웹소켓 프로토콜을 사용해 진행됩니다.
